# helper-docs

This is a Next.js application generated with
[Create Fumadocs](https://github.com/fuma-nama/fumadocs).

Run development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open http://localhost:3000 with your browser to see the result.

## Generating API Documentation

To generate documentation from the OpenAPI specification:

```bash
node scripts/generate-docs.mjs
```

This script uses fumadocs-openapi to convert the OpenAPI schema (located at `./public/openapi.json`) into documentation files in the `./content/docs/api-reference` directory.
