## Other platforms

Helper is a Next.js app, so follow the Next.js deployment guide for your platform of choice.

You will need to set up the following services. The environment variables may be set automatically if your platform has a built-in integration, or you may need to set them manually.

- [Supabase](https://supabase.com) (set up environment variables for `DATABASE_URL`, `POSTGRES_URL_NON_POOLING`, `SUPABASE_SERVICE_ROLE_KEY`, `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`)
 - `DATABASE_URL`: The main connection URL for your Supabase Postgres database.
 - `POSTGRES_URL_NON_POOLING`: The direct connection URL to your Supabase Postgres database (typically used for migrations if different from `DATABASE_URL`).
 - `SUPABASE_SERVICE_ROLE_KEY`: Your Supabase project's service role key.
 - `NEXT_PUBLIC_SUPABASE_URL`: Your Supabase project's public URL.
 - `NEXT_PUBLIC_SUPABASE_ANON_KEY`: Your Supabase project's public anon key.
